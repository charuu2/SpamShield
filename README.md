# 📌 Overview

SpamShield is an AI-powered spam detection application that combines the contextual understanding capabilities of **BERT** with the reliability of **TF-IDF ensemble learning** and **fuzzy pattern matching**.

Unlike traditional spam filters that depend solely on keywords, SpamShield understands the semantic meaning of messages, making it more effective against phishing attacks, scam messages, promotional spam, and disguised malicious content.

The project features a modern cybersecurity-inspired interface with real-time inference, confidence scoring, and explainable predictions.
